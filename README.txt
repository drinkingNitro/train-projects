Matrix - реализация класса "матрица", включающая методы для работы с объектом типа матрица: 
- сложение
- умножение
- транспонирование
- вычисление определителя
- решение СЛАУ методом Крамера
- Преобразование в строку (переопределенный __str__).

Дополнительно реализован класс MatrixError, который предназаначен для обработки ошибок, связанных с работой с основным классом.

Реализован класс SquareMatrix, являющийся наследником основного класса и предназначеный для работы с квадратными матрицами,
содержит метод для возведения матрицы в степень.

Файл /tests for Matrix.py/ содержит тестовые примеры для работы с классом Matrix.
